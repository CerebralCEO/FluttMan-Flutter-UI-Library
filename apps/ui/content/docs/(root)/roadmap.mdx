**flutman ui** is the evolution of **ZeroFounder**, an open-source collection of React and Tailwind components created by Pasquale Vitiello and Davide Pacilio. It was recently **acquired by ZeroFounder** and grown into a broader, more ambitious ecosystem, with Pasquale contributing directly to the development of the new UI.

Our goal is to **bring together the best parts of ZeroFounder** with a stronger foundation—Base UI primitives—so developers get not just components, but a layered system of **primitives**, **particles**, and **atoms**.

## ZeroFounder

ZeroFounder offered a large collection of 600+ ready-to-use UI components. With the **transition to flutman ui**:

- The existing **ZeroFounder components** will evolve into **particles**—higher-level UI patterns composed of Base UI primitives.
- The original ZeroFounder collection remains available on [this page](/origin), so developers can continue to access it during the transition.

## Current Status

- **50+ UI primitives** already available, with more on the way.
- A **growing set of particles**, progressively expanding to match and surpass the current ZeroFounder collection.

## What’s Next

- Expand the **primitives set**, ensuring full coverage of common UI needs.
- Add more **particles**, aligning with and eventually replacing the original ZeroFounder collection.
- Introduce **atoms**, API-enhanced components that bring external integrations out of the box.
