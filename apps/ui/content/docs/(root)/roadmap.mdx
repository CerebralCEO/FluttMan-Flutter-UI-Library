**FlutMan** is a zero-dependency, copy-paste UI component library for Flutter, inspired by **shadcn/ui**. We're building a comprehensive ecosystem of **components**, **particles**, and **atoms** for the Flutter community.

Our goal is to provide Flutter developers with a complete set of beautiful, accessible, and customizable UI components that can be easily added to any project using our CLI.

## Current Status

- **50+ UI components** already available, with more on the way.
- **FlutMan CLI** for easy component installation and project setup.
- A **growing set of particles** — higher-level UI patterns composed of multiple components.

## What's Next

### Short Term

- Expand the **components set**, ensuring full coverage of common UI needs.
- Add more **particles** — pre-assembled patterns like authentication forms, data tables, and date pickers.
- Improve **documentation** with more examples and use cases.
- Add **dark mode support** for all components.

### Medium Term

- Introduce **atoms** — API-enhanced components that integrate with external services.
- Add **animations and transitions** package for polished UI effects.
- Support for **desktop and web** Flutter platforms.

### Long Term

- **Visual editor** for building UIs with FlutMan components.
- **Figma plugin** to sync designs with code.
- **Community marketplace** for sharing custom components.

## Get Involved

- **Submit issues** for bugs or feature requests
- **Contribute components** following our design guidelines
- **Improve documentation** with examples and tutorials
- **Share feedback** to help shape the roadmap
