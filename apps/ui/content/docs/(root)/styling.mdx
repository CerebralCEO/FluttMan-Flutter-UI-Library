## Overview

FlutMan components are designed to work seamlessly with Flutter's built-in theming system. Each component accepts standard Flutter styling properties and can be customized to match your design system.

Our components follow Material Design 3 principles while remaining fully customizable. They use:

- **ThemeData** for global color schemes and typography
- **Widget-specific properties** for granular control
- **Composition patterns** for building complex UIs

## Installation

See [Get Started](/ui/docs/get-started) for installing with the CLI. When you add a component, it automatically integrates with your app's theme.

## Theming with ThemeData

Flutter's `ThemeData` is the primary way to customize the look and feel of your app. FlutMan components respect your theme's:

- **Color scheme** (primary, secondary, surface, background, error)
- **Typography** (headlines, body text, captions)
- **Shape** (rounded corners, borders)
- **Elevation** (shadows, overlays)

### Setting Up Your Theme

Define your theme in your `MaterialApp`:

```dart title="lib/main.dart"
MaterialApp(
  theme: ThemeData(
    colorScheme: ColorScheme.fromSeed(
      seedColor: Colors.blue,
      brightness: Brightness.light,
    ),
    useMaterial3: true,
  ),
  darkTheme: ThemeData(
    colorScheme: ColorScheme.fromSeed(
      seedColor: Colors.blue,
      brightness: Brightness.dark,
    ),
    useMaterial3: true,
  ),
  home: MyHomePage(),
);
```

## Component Styling

Each FlutMan component accepts standard Flutter styling properties:

### Colors

Most components accept a `color` or `backgroundColor` property:

```dart
FlutManButton(
  backgroundColor: Theme.of(context).colorScheme.primary,
  foregroundColor: Theme.of(context).colorScheme.onPrimary,
  onPressed: () {},
  child: Text('Primary Button'),
);
```

### Typography

Text styling is controlled through the theme or widget properties:

```dart
FlutManCard(
  child: Column(
    children: [
      Text(
        'Title',
        style: Theme.of(context).textTheme.headlineSmall,
      ),
      Text(
        'Body text',
        style: Theme.of(context).textTheme.bodyMedium,
      ),
    ],
  ),
);
```

### Shapes and Borders

Customize shapes using Flutter's `ShapeBorder` and `BorderRadius`:

```dart
FlutManCard(
  shape: RoundedRectangleBorder(
    borderRadius: BorderRadius.circular(16),
  ),
  child: Text('Custom rounded card'),
);
```

## Design Tokens

For consistency across your app, define design tokens as constants:

```dart title="lib/ui/theme/tokens.dart"
class DesignTokens {
  // Spacing
  static const double spacingXs = 4;
  static const double spacingSm = 8;
  static const double spacingMd = 16;
  static const double spacingLg = 24;
  static const double spacingXl = 32;

  // Border radius
  static const double radiusSm = 4;
  static const double radiusMd = 8;
  static const double radiusLg = 12;
  static const double radiusXl = 16;

  // Elevation
  static const double elevationNone = 0;
  static const double elevationSm = 2;
  static const double elevationMd = 4;
  static const double elevationLg = 8;
}
```

## Dark Mode

Flutter's theme system handles dark mode automatically. FlutMan components respect the current brightness setting:

```dart
MaterialApp(
  theme: ThemeData(
    colorScheme: ColorScheme.fromSeed(
      seedColor: Colors.blue,
      brightness: Brightness.light,
    ),
  ),
  darkTheme: ThemeData(
    colorScheme: ColorScheme.fromSeed(
      seedColor: Colors.blue,
      brightness: Brightness.dark,
    ),
  ),
  themeMode: ThemeMode.system, // Follows system setting
  home: MyHomePage(),
);
```

Components automatically adapt their colors when the theme changes.
