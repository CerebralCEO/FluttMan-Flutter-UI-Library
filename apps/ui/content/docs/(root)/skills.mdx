FlutMan provides structured knowledge for AI coding assistants — component APIs, composition patterns, styling conventions, and common pitfalls. When you use the **Copy Prompt** button on any component page, you get an AI-ready instruction that helps your assistant use FlutMan components correctly.

For example, you can ask your assistant to:

- _"Add a settings dialog with a form and save/cancel buttons."_
- _"Add a dropdown menu with grouped options."_
- _"Create a data table with pagination."_
- _"Build a toast notification for form submission errors."_

## Using the Copy Prompt Button

Every component page has a **Copy Prompt** button that generates an AI instruction:

```text
You are a Flutter expert. Open the terminal in the root of this project 
and execute the command: `flutman add button`. Once the component is 
downloaded into lib/ui/components/, review the Dart code and implement 
it in the current screen appropriately.
```

Copy this prompt and paste it to your AI assistant to get accurate, context-aware help.

## What's Included

The FlutMan documentation covers the full component surface:

### Component Knowledge

Reference guides for all components — imports, minimal patterns, inline code examples, composition rules, and common pitfalls. The assistant knows when to use Dialog vs Sheet vs Drawer, how to compose overlays, and how to structure forms.

### Styling Conventions

Flutter ThemeData usage, Material Design 3 integration, color scheme configuration, and design token patterns.

### CLI and Workflow

Full reference for installing components via the FlutMan CLI:
- `flutman add <component>` — Add a specific component
- `flutman init` — Initialize FlutMan in a project
- `flutman list` — See all available components

### Particle Examples

The documentation includes real-world composition examples for every component — so the assistant can produce production-realistic code, not just minimal stubs.

## How It Works

1. **Copy the prompt** — Click the **Copy Prompt** button on any component page
2. **Paste to your AI** — Share the prompt with your AI assistant (Claude, ChatGPT, Cursor, etc.)
3. **Get accurate code** — The assistant follows FlutMan patterns and generates proper Flutter code

## Best Practices for AI Prompts

When asking your AI assistant for help:

- **Be specific** — Mention the exact component name (e.g., "FlutManButton", "FlutManCard")
- **Provide context** — Share your current screen/widget structure
- **Use Copy Prompt** — Always start with the component's prompt for best results
